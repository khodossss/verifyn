        About Verifyn

        AI-Driven Fact-Checking at Scale

        Verifyn is an autonomous fact-checking engine. Every submission is processed by a ReAct AI agent that independently searches the live web, cross-references primary sources, and applies a structured 9-step verification methodology — without human intervention.

        This is not a keyword filter or a static database lookup. The agent reasons, searches, reads, and concludes — the same way an investigative journalist would, at machine speed. Built with LangGraph, FastAPI, and OpenAI.

        9-Step Methodology
        1. History Search — Check if this or a similar claim has been fact-checked before using embedding similarity search.
        2. Claim Extraction — Isolate the core verifiable assertion from the submitted text.
        3. Primary Source Search — Locate the original source of the claim via targeted web search.
        4. Date & Context Verification — Confirm when the event occurred and whether the framing is current.
        5. Lateral Reading — Search what independent outlets report about the same claim.
        6. Corroboration Count — Assess how many credible sources confirm or contradict the claim.
        7. Old Content Check — Detect recycled or out-of-context material being passed as new.
        8. Professional Fact-Checkers — Query Snopes, PolitiFact, Reuters Fact Check, AP, Full Fact, and AFP.
        9. Classification — Synthesise all evidence into a verdict with confidence score and structured reasoning.
